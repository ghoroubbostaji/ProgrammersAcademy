2020 Jan Python as the #1 programming language
Python has seen enormous growth in demand with no sign of slowing down. The programming language ranking site PYPL has ranked Python as the #1 Programming Languge. Also, Python has surpassed Java and became the 2nd most popular language according to GitHub repositories contributions.
2020 Sept Learnt to code during COVID-19
Research from digital transformation firm BoxBoat suggests that around one in four people spent time learning coding languages during the COVID-19 lockdown. The greatest motivations for people setting out to improve their skills were career development (55%), personal development (46%), and improving job search prospects (33%).
